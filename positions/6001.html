Senior Active Directory Administrator

Experience: 7-10 years | Location: Bangalore

Job Overview

The Senior Active Directory Administrator is a seasoned professional with extensive experience in designing, deploying, and managing Active Directory environments. This role requires in-depth knowledge of Windows Server platforms and a strong background in AD architecture, upgrade, and customization. The individual will play a crucial role in ensuring the stability, security, and performance of the Active Directory infrastructure.

Job Responsibilities

  • AD Design and Deployment: Design and deploy Active Directory environments on Windows Server 2008 R2/2012 R2/2016 platforms. Upgrade existing AD environments to meet the specified requirements.
  • AD Domain Services Expertise:Expertise in building new Active Directory domains and create comprehensive Low-Level Designs. Configure and manage Active Directory Federation Services (ADFS) including Low-Level Design creation.
  • AD Certificate Services (PKI):Expertise in building and configure Active Directory Certificate Services, ensuring a secure Public Key Infrastructure (PKI).
  • GPO Configuration and Management:Strong understanding of Group Policy Objects (GPO) and effectively configure and manage them.
  • OU Structure Planning: Develop and implement effective Organizational Unit (OU) structures, optimizing AD organization and administration.
  • AD Sites and Replication: Possess an in-depth understanding of AD sites and replication topology, ensuring efficient communication across the network.
  • Technical Notions and Expertise: Exhibit advanced technical knowledge in Wintel Backup, storage virtualization, and Active Directory backup and restore.
  • AD services: Strong knowledge on Network Time protocol, Account lockout issues, Delegation related issues, Dc health monitoring, FSMO transfer and Seize, Sysvol replication issues, DFSR ,FGPP , GSMA, Good understanding on Netmon logs , Certificates , Certificates renewal on ADFS.
  • Windows Server Administration: Install, configure, and provide support for Windows Servers. Conduct system maintenance, monitor performance, and ensure system security.
  • Architecting MS Windows OS Technology: Configure and architect MS Windows OS technologies including AD Forests, Domains, Trusts, DNS, DHCP, Group Policy, and Organizational Units.
  • Active Directory Upgrades and Troubleshooting: Plan and execute domain-level upgrades in Active Directory. Strong Experience on AD upgrade to Windows Server 2008 R2/2012 R2/2016. Troubleshoot and resolve GPO and Active Directory Federated Services issues.
  • Rights and Privileges Management: Review domain-level rights and privileges, making necessary changes to ensure security and compliance.
  • DNS and DHCP Support: Assist in the management and support of internal and external DNS systems, as well as internal DHCP architecture. Expertise in DHCP failover.

Qualifications:

  • Bachelor's degree in Computer Science, Information Technology, or a related field
  • [7-10] of hands-on experience in Active Directory administration.
  • Proficient in Windows Server 2008 R2/2012 R2/2016.
  • Expertise in AD, ADFS, PowerShell, PKI technologies.
  • Knowledge of Workplace environment control methods (Logon Scripts, Profiles, Policies, etc.).
  • Relevant certifications (e.g., MCSE) will be an advantage.
